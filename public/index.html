Little Commits

Get a print-out of each commit made to a repository.

Setup is simple. Just go to the 'Service Hooks' part of your repo's settings, and add the following URL as a WebHook URL:

http://little-commits.herokuapp.com/hook/YOUR_PRINT_CODE

Replace YOUR_PRINT_CODE with your direct print code. Find it here.

Your print code is not stored, and if you want you can reset it on the page where you found it.

If you want, you can check out the source code on GitHub.
